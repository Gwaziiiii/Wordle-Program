**Word Selection**: The program randomly selects a secret word from a pre-defined list of 50 options.

**User Input**: The user is prompted to guess a 5-letter word, with a total of 6 attempts allowed.

**Evaluation Logic**:
   
      -Green: Letters in the correct position are tracked and displayed.
    
      -Yellow: Letters present in the word but in the wrong position are identified.

**Feedback**: After each guess, the program displays the current state of the hidden word and lists the identified green and yellow letters.
